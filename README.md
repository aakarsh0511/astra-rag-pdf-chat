# Astra RAG PDF Chat

A Retrieval-Augmented Generation (RAG) application that enables users to query PDF documents using natural language. The project leverages LangChain, Astra DB, vector embeddings, and Large Language Models to provide accurate, context-aware responses from uploaded PDFs.

## Features

* PDF document ingestion
* Vector embedding generation
* Astra DB vector storage
* Semantic search and retrieval
* Context-aware question answering

## Tech Stack

* Python
* LangChain
* Astra DB (Vector Database)
* OpenAI
* Jupyter Notebook

## Usage

1. Upload PDF documents.
2. Generate embeddings and store them in Astra DB.
3. Ask questions in natural language.
4. Receive context-based answers from the documents.
